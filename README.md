# IoT chaos client

## Dependencies
  * Node
  * NPM
  * Docker

## Install
  * Clone this repository
  * npm i
  * npm start

## Development
  * **test:**  
  `npm run test`  

  * **lint:**  
  `npm run eslint`  

  * **build a new version:**  
  `npm version [major|minor|patch] [--force]`  
  *this assumes access to the container registry for the image being built*
